roche.io

javascript platformer

You must have a keyboard and a window width of more than 900 pixels to play. Please resize this window or come back later on a desktop or laptop computer.

About:

I'm learning to design browser-based games from the ground up. For added motivation, I decided to build everything here in public. What you see is the game so far.

Current High Score - JNM - 12.0 seconds

You can follow my development on Github.

Basic Controls:

  • [A, Left]: Move left
  • [D, Right]: Move right
  • [W, Up, Space]: Jump
  • [S, Down]: Stop movement
  • [Q, CapsLock]: Run
  • [Z]: Pause
  • [X]: Restart game
  • [C]: Respawn on current level
